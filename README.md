# Black-Scholes-Model
# Published in 1973 by Fisher Black, Robert Merton and Myron Scholes

**
        # Black Scholes Model has been developed to calculate the price of options and either Call option or Put option.
       
        # It is a Mathematical equation of pricing the stock.

        # The model is used to determine the fair price of stock options.
        
        # It is based on six variables suce as:**
                # - Type of option( Call or Put option)
                # - Underlying stock price (spot price)
                # - Time 
                # - Strike Price
                # - Volatility
                # - Risk-free rate
